# Global Edible Oil Price Forecasting Dashboard

## Objective

Forecast edible oil prices over the next four weeks using historical market data.

## Dataset

- Palm Oil
- Soybean Oil
- Sunflower Oil
- Canola Oil
- Crude Oil
- Shipping Cost Index
- USD Index

## Dashboard Features

- Historical Trends
- Forecasting
- KPI Monitoring
- Business Drivers
- Model Performance
- Pipeline Monitoring

## Technologies

- Python
- Streamlit
- Prophet
- Pandas
- Plotly
